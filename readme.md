## 🧰 Features

- Convert HTML/CSS to (React, React Native)
- Organize style
- Generate Jest
- Generate Storybook
- Realtime conversion mode
- Matrix loading animations :P

## 📈 Advantage

##### With short and simple html/css code creates a professional structure.

###### write this:

```html
<body>
  <div id="Home">
    <div id="Header">
      <h1 id="Title">Home Page</h1>
    </div>
  </div>
</body>
```

###### receive this:

```
source
├── components
│   ├── Header
│   │   ├── Header.jsx
│   │   ├── Header.stories.jsx
│   │   ├── Header.styled.js
│   │   ├── Header.test.jsx
│   │   └── index.js
│   └── Title
│       ├── Title.jsx
│       ├── Title.stories.jsx
│       ├── Title.styled.js
│       ├── Title.test.jsx
│       └── index.js
└── pages
    └── Home
        ├── Home.jsx
        ├── Home.stories.jsx
        ├── Home.styled.js
        ├── Home.test.jsx
        └── index.js

```
